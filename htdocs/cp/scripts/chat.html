About
WebChat is a useful application that allows a number of people on the World Wide Web to talk to one another simultaneously. The ability to chat on the Web can be a quick way to hold a virtual meeting.

Starting the WebChat
The WebChat is installed in your user directory. You can create links to the WebChat from any page in your site by adding a relative link to /cgi-bin/chat/chat.cgi.
For example, in your index.html page, you can add a link to the WebChat with the following line of html code:
<a href="cgi-bin/chat/chat.cgi">WebChat</a>
To go to the WebChat right now, click here.
To change the settings of the WebChat, edit configuration files in /cgi-bin/chat/ of your user directory.
